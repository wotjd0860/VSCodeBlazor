# VSCodeBlazor
Blazor Project with VSCode

1. To make blazor server (Terminal)
- dotnet new blazorserver -f net7.0

2. To run blazor server (Terminal)
- dotnet watch run

3. To stop blazor server (Terminal)
- Ctrl + C

git config --global user.name "John Doe"
